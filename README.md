#FICT Student Management System (Linked Lists)

A C++ application developed using **Microsoft Visual Studio 2022** that manages, filters, and updates student academic records utilizing custom dynamic Linked Lists.

## 🛠️ Tech Stack & Concepts
- **Language:** C++
- **IDE:** Microsoft Visual Studio 2022
- **Data Structures:** Custom Doubly/Singly Linked Lists, Struct Arrays
- **Core Concepts:** Pointer manipulation, dynamic memory allocation, file I/O operations, algorithmic data filtering

## 📋 Core Features
- **Dynamic Linked Lists:** Manages student records (`Student`, `Exam`, and `Subject` structs) with no duplicate entries allowed.
- **Data Transformation:** Automated batch processing to update legacy 7-digit phone numbers to 9-digit formats and format course prefixes onto IDs.
- **Advanced Filtering & Statistics:** Algorithms to parse trimester results, calculate running CGPA/GPA metrics, and isolate potential first-class honors candidates based on credit hour thresholds.
